# 🍕 Pizza Sales Dashboard (Tableau)

## 📊 Overview

This project presents an interactive Tableau dashboard analyzing pizza sales data.

## 🔗 Live Dashboard

👉 https://public.tableau.com/app/profile/likith.kasireddi/viz/TableauPizzaSalesDashboard

## 📈 Key Features

* Top 3 revenue generating pizza types
* Top 5 popular pizzas
* Revenue contribution analysis (Top 10)
* Order trends by hour
* Cumulative revenue over time
* Most ordered pizza size

## 🛠 Tools Used

* Tableau Public
* CSV Dataset
